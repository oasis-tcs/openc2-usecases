# Introspection

## Objective
This use case describes how an actuator reports its supported capabilities to an orchestrator
or management console.

## Scenario
TBS

## OpenC2 JSON Command

```
{
    "action": "query",
    "target": {
        "openc2": ["versions"]
    }
}
```

## Response

```
{
    "status": "200",
    "results": {
        "versions": ["/oasis-open.org/openc2/v1.0/openc2-lang"]
}
```
